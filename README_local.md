# Sahaya Assitency

## Setup Instructions

### Backend
1. Place `serviceAccountKey.json` inside the `backend` folder.
2. Run the following commands:
   ```bash
   cd backend
   npm install
   node server.js
   ```

### Frontend
1. Add `firebaseConfig.ts` manually.
2. Run the following commands:
   ```bash
   cd frontend
   npm install
   npx expo start
   ```

> **Note:** Replace `BASE_URL` in the frontend code with your local network IP before running.
